What is this?

This site finds your location and shows it on a map along with the locations of any heritage plaques nearby.

Why?

It was built as part of @historyhackday 2011 by @fredkingham and @grahamlyons and the data comes from the Open Plaques project.

How does it work?

It was originally built in PHP with the Zend framework and got the user's location data from Twitter. It has since been rebuilt in Python using Flask and the now more mature geolocation api in the browser.

Can I see?

The code powering the site is available on Github if you want to have a look at it.
